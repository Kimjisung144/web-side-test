홈화면

오버로딩 같은경우 같은 클래스 안에서 메소드 이름은 같지만, 매개변수(parameter)의 개수나 타입이 다른 메소드를 여러 개 정의하는 것이다.
조건은 메소드 이름: 같아야 함
매개변수(파라미터) 리스트: 달라야 함 (개수, 타입, 순서 중 하나라도 달라야 함)
리턴 타입: 달라도 되지만, 매개변수만 다르면 성립
class Calculator {
int add(int a, int b) {
return a + b;
}
double add(double a, double b) {
return a + b;
}
int add(int a, int b, int c) {
return a + b + c;
}
}

오버라이딩 같은경우 상속 관계에서 부모 클래스의 메소드를 자식 클래스에서 재정의하는 것
메소드 이름: 같아야 함
매개변수 리스트: 완전히 동일해야 함
리턴 타입: 부모와 같거나, 자식 타입(covariant return) 가능
class Vehicle {
void sound() {
System.out.println("교통수단 소리를 냅니다");
}
}
class Car extends Vehicle {
@Override
void sound(){
System.out.println("부우우웅~");
}
}
class Bike extends Vehicle {
@Override
void sound() {
System.out.println("브르르르~");
}
}
public class Main {
public static void main(String[] args) {
Vehicle myCar = new Car();
Vehicle myBike = new Bike();
myCar.sound(); // 부우우웅~
myBike.sound(); // 브르르르~
}
}
